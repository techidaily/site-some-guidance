### 1\. Transcode MP4 to a Smaller-Sized FLV, WMV, HEVC

There is a bewildering array of video formats besides MP4 - AVI, WMV, FLV, MOV, MKV, etc for compressing files, - and on top of that, how these formats stack up against each other, especially in terms of providing smaller file size than MP4? Which one is the best format to [shrink and transcode MP4](https://tools.techidaily.com/macxdvd/products/) to? Below we list two video formats with file size smaller than MP4\. 

**â€¢ WMV:** The file format is a tightly compressed format that requires a minimal amount of storage space, thus dramatically decreasing the size and increasing load time. It's proved the file size of MP4 is **reduced by 60%** after converting MP4 to WMV. But you must know WMV has bad compatibility with non-windows platforms, like iPhone, iPad and other Apple devices, and Linux players. This is way WMV isn't the standard web format. 

**â€¢ FLV:** It's a suitable format for web video. It's viewable by over 90% web visitors, small in size and easly to be downloaded. So it's workable to compress MP4 to smaller FLV for web upload. If you hope to decrease MP4 file size, especially for uploading to YouTube, Facebook, etc, it's advised to convert MP4 to smaller sized FLV with this [MP4 video converter](https://tools.techidaily.com/macxdvd/products/).

**â€¢ HEVC:** HEVC is a format developed to take place of MP4 H.264 to store high-definition videos. It can save 50% of space compared with H.264 at the same quality. Therefore, you can compress MP4 videos to HEVC to downsize the video, especially the source MP4 is in HD or 4K resolutions. You can select MP4 H.265 as the output format, or change the video codec to H.265 in the video parameter settings. Currently, HEVC is supported in major video editors, players, websites, web players, converters etc.

### #2\. QuickTime Player 

For Mac users seeking a quick and simple way to compress MP4 videos on Mac without the need for additional software, QuickTime Player serves as a handy tool. Its ease of use and availability as a built-in application make it convenient for basic compression needs. Still, users requiring more nuanced control over compression settings might find its options limited.

The compression process in QuickTime involves navigating to the "File" menu, selecting "Export As," and then choosing from a few preset options such as 4K, 1080p, 720p, or 480p, each utilizing either the H.264 or HEVC (H.265) codec for compression. The resolution options vary from up to 3840 x 2160 for 4K down to 640 x 480 for 480p. The "Use HEVC" option, especially for 1080p resolution or higher, employs the HEVC (H.265) codec, known for its superior compression efficiency compared to H.264, potentially offering better results in reducing file sizes while retaining quality.

However, Apple's QuickTime doesn't provide extensive customization options like adjusting specific bit rates or other detailed settings, which might limit users looking for more precise control over their compression settings. QuickTime's compression settings don't always give precise information about the final resolution or data rate of the compressed video. Apple doesn't explicitly clarify whether these settings strictly depend on the original file or if there's some automatic adjustment.
